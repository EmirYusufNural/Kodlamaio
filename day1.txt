1] Python’da Numerik Veri;
+ Python’da üç tür numerik yani sayısal veri tipi vardır; int, float, complex.
+ Integer veya tamsayı veri tipi, sınırsız uzunlukta pozitif veya negatif tam sayıları ifade etmektedir.
+ Float veya ondalıklı sayı veri tipi, noktadan sonra bir veya daha fazla ondalık sayı içeren pozitif veya negatif sayıları ifade etmektedir. Float, ayrıca içinde e veya E barındıran tüm bilimsel sayıları tanımlarken kullanılır.
+ + Python da karmasik sayilar yerine complex sayilari kullaniriz i yerine j degeri kullanilir(Matematikte, a +b i şeklinde ifade edilirler. Burada a sayısının gerçel (real) kısmını, b ise sanal (imaginary) kısmını ifade etmektedir. i ise √−1 anlamındadır.)
Python’da String Veri Tipi
Python’daki string değerler tek tırnak işareti veya çift tırnak işareti içinde ifade edilmektedir. Diğer bir ifade ile ‘Merhaba’ ve “Merhaba” Python için aynıdır.
Bir değişkene string değer atama, değişken adı ve ardından eşittir işaretiyle yapılır.
Istersek 3 tane """ """ arasinda istedigimiz sekilde kodu duzenleyebiliriz.

+ Text Type:	str  x = "Hello World"	str
+ Numeric Types:	int, float, complex    **x = 20	int,x = 20.5	float,x = 1j	complex**

+ Sequence Types:	list, tuple, range  **x = ["apple", "banana", "cherry"]	list,x = ("apple", "banana", "cherry")	tuple	,x = range(6)	range**
+ Mapping Type:	dict **x = {"name" : "John", "age" : 36}	dict**
+ Set Types:	set, frozenset **x = {"apple", "banana", "cherry"}	set	,x = frozenset({"apple", "banana", "cherry"})	frozenset**
+ Boolean Type:	bool **x = True	bool**
+ Binary Types:	bytes, bytearray, memoryview **x = b"Hello"	bytes / x = bytearray(5)	bytearray	/ x = memoryview(bytes(5))	memoryview**
+ None Type:	NoneType **x = None	NoneType**
